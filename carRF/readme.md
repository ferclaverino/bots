### How to connect nRF24L01+ Pins

|nRF24L01+|Arduino (Uno, Nano)|
|---|---|
|GND|GND|
|VCC|3.3V|
|CE|D7|
|CSN|D8|
|SCK|D13|
|MOSI|D11|
|MISO|D12|
|IRQ|-|
